Iterative Implementation of DFS

marked = [False] * G.size()
def dfs(G,v):
    stack = [v]
    while len(stack) > 0:
        v = stack.pop()
        if not marked[v]:
            visit(v)
            marked[v] = True
            for w in G.neighbours(v):
                if not marked[w]:
                    stack.append(w)

Time complexity: O(V+E)O(V+E)O(V+E); VVV = no. of vertices EEE = no. of edges

  • More generalizable and extensible than Recursive Implementation of DFS. As in, it can be tweaked to incorporate other algorithms.
  • Using Iterative Implementation, you can explicitly specify the stack size. While recursive approach uses the OS' default stack size, which is still typically small - and thus can cause stack overflow.
