Hashmap

Hashmaps can be implemented using the Map data structure.

Creating

let hashmap: Map<KeyType, ValueType> = new Map();

Adding values

hashmap.set(key, value);

Getting values

val = hashmap.get(key)

Check if key exists

hashmap.has(key)

Deleting

hashmap.delete(key)

Size

hashmap.size

Clearing

hashmap.clear()
