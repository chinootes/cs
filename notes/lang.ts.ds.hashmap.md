Hashmaps can be implemented using the Map data structure.

## Creating

```ts
let hashmap: Map<KeyType, ValueType> = new Map();
```

## Adding values

```ts
hashmap.set(key, value);
```

## Getting values

```ts
val = hashmap.get(key)
```

## Check if key exists

```ts
hashmap.has(key)
```

## Deleting

```ts
hashmap.delete(key)
```

## Size

```ts
hashmap.size
```

## Clearing

```ts
hashmap.clear()
```
